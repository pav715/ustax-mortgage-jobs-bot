# US Tax Mortgage Jobs Bot

Automated LinkedIn job scraper for **US Tax & Mortgage / Loan Servicing** roles in India.

Posts to Telegram channel **@US_mortgage_jobs_1** hourly (8 AM – 11:30 PM IST).

## Roles covered

- Mortgage Tax / Tax Servicing
- Loan Servicing / Mortgage Operations
- Document Indexing / Credit Pack (Black Knight style)
- Wells Fargo / MSP / Escrow / HMDA / 1098 roles
