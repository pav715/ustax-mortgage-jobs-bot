# US Tax Mortgage Jobs Bot

LinkedIn bot for **US Tax + Mortgage / Loan Servicing** roles — aligned with
Wells Fargo & Black Knight Financial Services experience (loan lifecycle, doc indexing,
credit pack, mortgage servicing, mortgage tax).

## Setup (GitHub)

1. Create new repo on GitHub (public recommended = unlimited Actions minutes)
2. Push this folder to `main`
3. Add **Secrets** in repo Settings → Secrets → Actions:
   - `BOT_TOKEN` — same Telegram bot token as other bots
   - `CHAT_ID` — your new channel ID (send me to update if needed)
4. Add bot as **admin** to the Telegram channel (post messages permission)
5. Run **US Tax Mortgage Jobs Bot (Manual)** once from Actions tab to test

## Schedule (IST)

- **8:00 AM – 11:30 PM** — hourly (17 schedulers)
- Off: 11:30 PM – 8:00 AM

## Search keywords (LinkedIn)

Mortgage Tax, Mortgage Servicing, Loan Servicing, Mortgage Operations, US Mortgage

## Filter (resume-based)

Matches roles like: Process Associate, Loan Servicing, Document Indexing, Credit Pack,
Black Knight / MSP, Wells Fargo mortgage, Escrow, HMDA, 1098, Loss Mitigation, etc.

Rejects: Indian GST/Income Tax, recruiters, pure IT/software, construction roles.
